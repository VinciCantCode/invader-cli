## Overview

A simple terminal-based Space Invaders game written in Go.

## How to Play

- Choose a board size (Small / Medium / Large).
- Choose difficulty (Easy / Medium / Hard).
- Move the ship at the bottom and shoot down all aliens before they reach you.

### Controls

- Arrow keys: Move
- Space: Shoot
- P: Pause / resume
- F: Restart
- Q / Esc: Quit

## Build & Run

```bash
go build -o invader
./invader
```
